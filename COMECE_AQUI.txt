🎯 O SISTEMA FAZ O QUÊ?
═══════════════════════════════════════════════════════════════════════════════

✅ QUANDO O BOT CONECTA:
   → Monitor de vencimentos inicia automaticamente

✅ DIARIAMENTE ÀS 08:00:
   → Conecta à API Google Contacts
   → Lista todos os contatos
   → Extrai datas dos nomes (formato: 🔍 21/03/2026 | NOME)
   → Verifica quais venceram HOJE
   → Envia mensagem de aviso para cada vencimento
   → Marca para não enviar novamente naquele dia

✅ RESULTADO:
   → Cliente recebe: "🚨 AVISO DE VENCIMENTO - Seu acesso vence HOJE!"
   → Apenas 1 mensagem por cliente por dia
   → Automático, sem ação manual
